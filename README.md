# SootAL-Ard
منصة تواصل اجتماعي بيئية متكاملة مبنية باستخدام تقنيات الويب الحديثة (HTML5, CSS3, JavaScript ES6) مع الاعتماد الكامل على Firebase كقاعدة بيانات لحظية (Firestore) ونظام مصادقة (Auth). يتميز المشروع بنظام ردود متطور (Nested Replies)، تفاعلات لحظية، ونظام إشعارات ذكي لتنبيه المستخدمين بالتفاعلات على منشوراتهم.
